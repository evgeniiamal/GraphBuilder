## GraphBuilder

## Описание
Приложение на Python для построения графиков математических функций. Пользователь вводит аналитическое выражение, после чего программа вычисляет значения функции и отображает её график в окне с использованием библиотеки Pygame.
Проект разработан в учебных целях для изучения основ компьютерной графики, обработки математических выражений и работы с библиотекой Pygame.

## Основные возможности
- построение графиков математических функций
- ввод функции пользователем
- отображение координатных осей
- вычисление значений функции с использованием модуля math
- визуализация результатов в графическом окне

## Используемые технологии
- Python 3
- Pygame
- math

## Принцип работы
1. Пользователь вводит математическое выражение
2. Программа вычисляет значения функции на заданном интервале
3. Полученные координаты преобразуются в экранные
4. График отображается в окне приложения
